## 📌 프로젝트 개요

이 프로젝트는 **HTML**과 **CSS**로 만든 **넷플릭스 클론**입니다.  
주요 구성:

- 상단 로고 및 프로필 포함 내비게이션 바
- 재생/저장 버튼이 있는 메인 배너
- 가로로 스크롤되는 영화 카드 리스트
